# Menu Bar Icon Requirements

Use a monochrome template icon for the macOS menu bar.

- Format: PDF vector preferred, transparent background.
- Style: single color, no gradients, no shadows.
- Canvas: draw for a 18 x 18 pt area.
- Stroke weight: around 1.5 to 2 pt so the icon stays readable at small size.
- Padding: leave 1 to 2 pt inner padding on every side.
- Export: provide one asset that works as a template image. The system will tint it automatically for light and dark menu bars.

If you make a PNG instead of PDF:

- Provide at least 36 x 36 px.
- Keep it pure black on transparent background so it can be marked as a template image later.

Until a custom icon is ready, the app uses an SF Symbol in the menu bar.
